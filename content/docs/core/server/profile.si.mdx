V razdelku Profil lahko prilagodite nastavitve računa.

## Nastavitve računa

1. **Spremeni E-pošto**: Posodobite svoj e-poštni naslov.
2. **Spremeni Geslo**: Posodobite svoje geslo.
3. **Spreminjanje Avatarja**: Posodobite svojo profilno sliko.

## Dvofaktorska Avtentikacija (2FA)

Povečajte varnost računa tako, da omogočite dvofaktorsko preverjanje pristnosti z aplikacijo Google Authenticator.

## Dostop Preko API/CLI

Za dostop do API/CLI lahko ustvarite nov ključ API.

1. **Generirajte API Ključ**: Ustvarite nov API ključ.
2. **Kopiraj API Ključ**: Kopirajte ustvarjeni API ključ za uporabo.
3. **Pogled API Dokumentacije**: Dostopajte do API dokumentacije Swagger za podrobno uporabo API-ja.
