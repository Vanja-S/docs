---

import { Step, Steps } from 'fumadocs-ui/components/steps';

Za ponastavitev gesla sledite tem korakom:

<Steps>
<Step>
Vpišite se v svoj VPS strežnik.
 
</Step>
    <Step>
    Zaženite spodnji ukaz, da dobite ID kontejnerja za Dokploy-ovo namestitev.

    ```bash
    docker ps
    ```
    </Step>
    <Step>
    Zaženite spodnji ukaz za odprtje lupine v Dokploy kontejnerju.

    ```bash
    docker exec -it <container-id> bash -c "pnpm run reset-password"
    ``` 
    </Step>
    <Step>
    Prikazalo se bo naključno geslo. Kopirajte ga in ga uporabite za ponovni dostop do nadzorne plošče.
    </Step>
</Steps>
